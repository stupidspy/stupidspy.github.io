添加数值千分位符号

在页面中，遇到数值的时候，要记得添加千分位符

如：
100.00 -> 100.00
1000.00 -> 1,000.00
1000000.00 -> 1,000,000.00

一、针对正整数

/*
 * 为正整数添加千分位符号（仅限正整数）
 * @param num (Number或者String)
 * @type String
 * */
function numSetSymbol (num){
    var result = num.toString();
    for (var i = 0; i < Math.floor((result.length-(1+i))/3); i++)
    {
        result = result.substring(0,result.length-(4*i+3))+','+result.substring(result.length-(4*i+3));
    }
    return result;
    /*返回值为字符串*/
}

二、不限数值类型

/**
 * 将数值四舍五入后格式化.（不限数值类型）
 *
 * @param num 数值(Number或者String)
 * @param cent 要保留的小数位(Number)
 * @param isThousand 是否需要千分位 0:不需要,1:需要(数值类型);
 * @return 格式的字符串,如'1,234,567.45'
 * @type String
 */
function formatNumber(num,cent,isThousand) {
    num = num.toString().replace(/\$|\,/g,'');

    // 检查传入数值为数值类型
    if(isNaN(num)){
        num = "0";
    }

    // 获取符号(正/负数)
    sign = (num == (num = Math.abs(num)));

    num = Math.floor(num*Math.pow(10,cent)+0.50000000001); // 把指定的小数位先转换成整数.多余的小数位四舍五入
    cents = num%Math.pow(10,cent);       // 求出小数位数值
    num = Math.floor(num/Math.pow(10,cent)).toString();  // 求出整数位数值
    cents = cents.toString();        // 把小数位转换成字符串,以便求小数位长度

    // 补足小数位到指定的位数
    while(cents.length<cent){
        cents = "0" + cents;
    }

    if(isThousand) {
        // 对整数部分进行千分位格式化.
        for (var i = 0; i < Math.floor((num.length-(1+i))/3); i++)
            num = num.substring(0,num.length-(4*i+3))+','+ num.substring(num.length-(4*i+3));
    }

    if (cent > 0)
        return (((sign)?'':'-') + num + '.' + cents);
    else
        return (((sign)?'':'-') + num);
}

//用法1
 function example1(){
     return new Promise(function (resolve, reject) {
         resolve(param);
     });
 }
 example1().then(function(param){
     //执行example1之后...
 }

//用法2
function example2(){
    var promise = $.Deferred();
    //操作
    ...
    promise.resolve(param);
    return promise;
}
//函数和回参一一对应
$.when(example2(),...).done(function(param,...){

//用法3
var promise = new Promise(function(resolve, reject) {
  if (/* 异步操作成功 */){
    resolve(value);
  } else {
    reject(error);
  }
});

promise.then(function(value) {
  // success
}, function(value) {
  // failure
});

//用法4
var getJSON = function(url) {
  var promise = new Promise(function(resolve, reject){
    var client = new XMLHttpRequest();
    client.open("GET", url);
    client.onreadystatechange = handler;
    client.responseType = "json";
    client.setRequestHeader("Accept", "application/json");
    client.send();

    function handler() {
      if (this.status === 200) { 
              resolve(this.response); 
          } else { 
              reject(new Error(this.statusText)); 
          }
    };
  });

  return promise;
};

条形图中的两个label

一般情况下，每个对象有一个单独的label，但是当你遇到需要两个分开label的时候（我也不理解为什么name不直接用样例legend），好吧，既然设计稿长这样，那就做这样吧。

首先，条形图默认有一个label，我们可以对它进行设置，在图形上显示name或者value，或者name+value，但是无法显示两个分离的label，这个时候，我能想到的解决方法就是，用其它配置，伪装成label的样子。

偶就是这么机智~

因为legend和tooltip需要用，所以我最后选中了markPoint，markPoint通常用于显示‘最大值’‘最小值’或者‘平均值’等特殊值，鉴于它能在页面上显示，那就用它了。

PS:用markPoint显示的前提是，要显示的value必须是特殊值。在这个例子中两个对象是单个数据，所以设置为‘max’还是‘min’都OK。

需要处理的问题是：markPoint有默认的symbol：热气球，试了为空，不行哦，真是任性，既然markPoint一定要有背景图，而我又不想要背景图，咋办呢
那就给他一张好了，设置透明背景图就OK了!

然后把文字样式改成对应的，就Everything is OK！

配置长这样：
